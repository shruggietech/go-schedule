# Installing go-schedule

go-schedule ships per-platform. Pick the guide for your operating system:

- **[Windows](INSTALL-windows.md)** — the `.msi`, the Windows service, `PATH`,
  upgrading, and uninstalling.
- **[Linux](INSTALL-linux.md)** — the release archive, systemd registration,
  and data paths.
- **[macOS](INSTALL-macos.md)** — the desktop bundle versus headless, launchd,
  and the boot-persistence caveat that catches people out.

Each guide is self-contained. Once installed, the [`gosched` command
reference](cli.md) covers every command and flag.
